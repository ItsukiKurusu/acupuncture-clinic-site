# ブログ投稿ガイド

VSCode上で新しいブログ記事を投稿する手順を説明します。

## 記事の作成方法

### 1. 新しいMarkdownファイルを作成

`posts/` ディレクトリに新しい `.md` ファイルを作成してください。

**ファイル名の例:**
- `posts/first-time-acupuncture.md`
- `posts/acupuncture-symptoms.md`
- `posts/beauty-acupuncture.md`

### 2. フロントマター（メタデータ）を追加

ファイルの先頭に以下の形式でメタデータを追加してください。

```yaml
---
title: "記事のタイトル"
date: "2025-01-23"
excerpt: "記事の概要（最大150文字程度）"
coverImage: ""
author: "記者の名前"
tags: ["タグ1", "タグ2", "タグ3"]
---
```

**メタデータの説明:**
- `title`: ブログ記事のタイトル
- `date`: 投稿日（YYYY-MM-DD形式）
- `excerpt`: 記事の簡潔な説明。トップページやブログ一覧で表示されます
- `coverImage`: カバー画像のURL（オプション。空にしても可）
- `author`: 記事の著者名
- `tags`: 複数のタグを配列形式で指定（検索やカテゴリー分けに使用）

### 3. 記事の本文を作成

フロントマターの後に、Markdown形式で記事を書きます。

```markdown
---
title: "鍼治療は痛い?初めての方の不安を解消"
date: "2025-01-23"
excerpt: "「鍼治療は痛そう」「怖い」と思っている方も多いのではないでしょうか。"
author: "鍼灸HANE"
tags: ["鍼灸", "初心者"]
---

## 見出し2

段落1

### 見出し3

段落2

- リスト項目1
- リスト項目2

1. 番号付きリスト1
2. 番号付きリスト2

**太字**、*斜体*、`コード`
```

## Markdown記法について

一般的なMarkdown記法が使用できます：

- `# 見出し1`
- `## 見出し2`
- `### 見出し3`
- `**太字**`
- `*斜体*`
- `` `コード` ``
- `[リンクテキスト](URL)`
- `- リスト項目`
- `1. 番号付きリスト`

## ファイルを保存して完了

ファイルをVSCodeで保存すると、以下の場所に自動的に反映されます：

1. **トップページ** - 最新3件の記事がプレビューされます
2. **ブログページ** (`/blog`) - 全記事が一覧で表示されます
3. **ブログ記事ページ** (`/blog/[slug]`) - 個別記事として表示されます

## 記事のスラッグについて

ブログ記事のURLは、ファイル名から自動的に生成されます。

**例:**
- `posts/first-time-acupuncture.md` → `/blog/first-time-acupuncture`
- `posts/acupuncture-symptoms.md` → `/blog/acupuncture-symptoms`

## 記事を修正する場合

1. `posts/` ディレクトリのファイルを開く
2. 内容を編集
3. ファイルを保存

変更が自動的に反映されます。

## デプロイについて

変更をWebサイトに反映させるには、GitHubにプッシュしてください：

```bash
git add -A
git commit -m "新しいブログ記事を追加: 記事のタイトル"
git push
```

## トラブルシューティング

**記事が表示されない場合：**
- ファイルが `posts/` ディレクトリにあるか確認してください
- ファイル拡張子が `.md` であることを確認してください
- フロントマターのフォーマットが正しいか確認してください

**レイアウトが崩れている場合：**
- Markdown記法を確認してください
- YAML形式のフロントマターのインデントを確認してください
